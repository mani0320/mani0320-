HTML

HTML (Hypertext Markup Language) is the standard language used to create and structure content on the web. It defines the structure of web pages by using a system of tags or elements that describe the different parts of a webpage, such as headings, paragraphs, links, images, and more.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation or visual appearance of a web page written in HTML. It controls the layout, colors, fonts, spacing, and overall design of the webpage.

JavaScript

JavaScript is a high-level, dynamic programming language primarily used for creating interactive and dynamic behavior on websites. It enables developers to add functionality to web pages, such as responding to user actions, manipulating HTML content, and controlling multimedia.
